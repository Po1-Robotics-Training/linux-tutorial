# 如何使用Linux

---

## 什麼是Linux?

- Linux 是一個作業系統，他的功能是
	1. 管理硬件 <!-- element frag="1"-->
	2. 管理軟體 <!-- element frag="2" -->
	3. 為用戶提供使用層面 <!-- element frag="3" -->

負責提供基本使用層面的是Coreutils。 <!-- element frag="4" -->
